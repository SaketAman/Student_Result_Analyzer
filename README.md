# Student Result Analyzer Dashboard

## Project Overview

The **Student Result Analyzer Dashboard** is an interactive dashboard developed to analyze and visualize the academic performance of students. The project is based on the results of **50 students** across **12 subjects** and provides meaningful insights into student performance using KPIs, charts, and filters.

The dashboard helps teachers, administrators, and academic coordinators monitor overall performance, identify top-performing students, analyze subject-wise results, and compare performance across different categories such as gender and stream.

---

## Objectives

* Analyze the academic performance of students.
* Calculate overall grades, average marks, and class positions.
* Identify pass and fail statistics.
* Compare performance across different streams and genders.
* Visualize subject-wise performance using interactive charts.
* Enable quick decision-making through data-driven insights.

---

## Dataset

The dataset contains information for **50 students**, including:

* Student Name
* Gender
* Stream
* Marks in 12 Subjects
* Total Marks
* Average Marks
* Grade
* Position
* Pass/Fail Status
